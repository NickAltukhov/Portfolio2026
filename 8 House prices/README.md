# Предсказание цен на жильё

## Описание

Файл "house_prices.ipynb" проводит EDA, предобработку и моделирование для предсказания цен на жильё (Kaggle датасет, содержащий информацию о различных характеристиках недвижимости и их ценах). Используются: feature engineering, пайплайны, модели машинного обучения (линейные, бустинг), SHAP для интерпретации.

Данные: train.csv/test.csv (1460 записей, 81 признаков).

## Основные шаги

- EDA: describe, распределения, корреляции, тесты нормальности, визуализации (гистограммы, boxplots, heatmaps)  
- Предобработка: Импутация (IterativeImputer), encoding (OneHot, Label, CatBoostEncoder), scaling (RobustScaler, QuantileTransformer), feature selection (RFECV, SelectKBest)  
- Пайплайны: Пайплайн с transformers, обработка выбросов (IsolationForest, boxcox)  
- Модели: LinearRegression, Lasso/Ridge/ElasticNet, CatBoost/XGBoost/LightGBM
- Кросс-валидация: KFold, cross_val_score (RMSE)
- Предсказания на test, submission.csv, сохранение модели (pickle)  

## Результаты

- Лучший полученный результат - 0.12196 RMSE (проверялось настоящей выборкой на сайте Kaggle), что является лучшим результатом в сравнении с результатами, полученными во время прохождения соревнования в 2016 году. 
- Самая важная фича - "Общее качество * площадь застройки"
