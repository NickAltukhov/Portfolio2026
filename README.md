# Портфолио. Николай Алтухов

Портфолио содержит некоторые мои пет-проекты по анализу данных:
1. Нахождение наиболее близкой даты звонка при сопоставлении двух выгрузок из разных программ  и проверка нарушений ФЗ-230
2. Преобразование неструктурированной информации со всех листов из всех файлов, находящихся в папке, в табличный формат
3. Парсинг
4. EDA и базовая статистика (Netflix датасет)
5. Выполнение A/B теста и SQL-запросов
6. Реализация нечеткого сравнения
7. Кластеризация и множественное предсказание
8. Предсказание цен на жильё
9. Другое
