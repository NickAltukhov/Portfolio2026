# EDA и базовая статистика (Netflix датасет)

## Описание

Jupyter Notebook (netflix_eda.ipynb) проводит разведочный анализ данных (EDA) датасета Netflix с фильмами и сериалами/шоу. Анализируются распределения, корреляции, топ-листы, визуализации и статистические тесты.

Данные: netflix_titles.csv (8807 записей).

## Основные шаги

- Импорт библиотек (pandas, numpy, seaborn, matplotlib, scipy.stats, wordcloud, re)  
- Загрузка и обзор данных
- Анализ распределений: типы (Movie/TV Show), годы выпуска, рейтинги, страны  
- Обработка длительности: разделение на минуты/сезоны, удаление аномалий  
- Визуализации: гистограммы, boxplots, heatmaps корреляций, облака слов
- Статистические тесты: нормальность, корреляция  
- Топ-листы: страны, режиссеры, актеры, жанры  
- Доверительные интервалы для длительности  

## Результаты

- Некоторые из инсайтов: США лидирует по контенту, тренд на снижение длительности, средняя длительность фильмов 99-100 мин (95% доверительный интервал), сериалы в среднем имеют 1.7-1.8 сезона (95% доверительный интервал).
- Различные визуализации (распределения, топ-10, облака слов).
